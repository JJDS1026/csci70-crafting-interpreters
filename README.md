## Functionality
The group chose to implement set 1.

### Do-while loop
```
var x = 2;
do {
    print x;
    x = x + 1;
} while(x < 12)
```

### Support the bang operator for numbers.
```
print !0;
var x = 1;
print !x;
```

### Implement “ASMD”
```
print 5 * 3 + 2;
```

### Left bit shift
```
print 5 << 2;
var x = 1;
print x << 5;
```
